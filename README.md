# SCG-Project :moneybag:

Work in progress :warning:

Script Python to analyze variances data

## Prerequisites

1. Python
2. pipenv

## How to use it

1. Install the dependencies: `pipenv install`
2. Activate the virtual enviroment: `pipenv shell`
3. Run the app: `python main.py`
